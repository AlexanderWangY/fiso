# fiso

> WARNING: This is still just a pet project for me to learn Rust, so don't use it for anything critical. PRs are welcome!

**fiso** (filesystem organizer) is a fast and flexible CLI tool for cleaning and restructuring your filesystem. It helps you find duplicate files, sort cluttered directories, and automatically move or organize files according to rulesets you define.

You can preview dry-runs for safety and also see auditable logs for actions fiso runs on your filesystem.
